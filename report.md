# Test Cases and Results

## 1. Homepage Loading
Status: Passed

Observation:
Homepage loads successfully and products are visible.

---

## 2. Navigation Menu Testing
Status: Failed

Observation:
Clothing and Accessories categories are not working properly.

Recommendation:
Fix navigation links or category filtering functionality.

---

## 3. Search Functionality Testing
Status: Failed

Observation:
Search list/results are not working properly.

Recommendation:
Fix search filtering and result display logic.

---

## 4. Add to Cart Functionality
Status: Partially Passed

Observation:
Products are added to cart and cart numbers increase correctly.
However, add-to-cart confirmation message is not displayed.

Recommendation:
Display success notification/message after adding product to cart.

---

## 5. Remove Cart Item Testing
Status: Failed

Observation:
Remove option is not available in the cart.

Recommendation:
Add remove/delete functionality for cart items.

---

## 6. Mobile Responsive Testing
Status: Failed

Observation:
Website is not fully responsive on mobile devices.

Issues Found:
- Layout alignment problems
- UI not adjusted properly for smaller screens

Recommendation:
Improve responsive CSS styling for mobile devices.

---

## 7. Order Placement Testing
Status: Passed

Observation:
"Order Placed Successfully" message is displayed correctly.

---

# Conclusion

The website works partially across browsers and devices. Core functionalities like homepage loading and order placement work properly. However, multiple issues were identified in navigation, search functionality, cart management, and mobile responsiveness. Improvements are recommended for better user experience and compatibility.
